# 适用于Telegram的New bing机器人
## 使用前先配置config.py文件

进文件查看具体配置

---

## 人格配置

使用前先配置人格
prompt/master.txt是主人对话时的人格
prompt/user.txt是其他人对话的人格

---

## 配置module/cookie.json文件
1.浏览器安装Cookie-Editor扩展.
2.访问New Bing.
3.在页面中点击扩展.
4.点击扩展右下角的Export-Export as JSON
5.将复制的内容粘贴到对应的cookie.json文件.

---

## 部署
用python运行main.py
